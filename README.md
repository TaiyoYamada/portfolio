## 技術スタック

### フロントエンド

| 技術                                                            | 用途                                                       |
| --------------------------------------------------------------- | ---------------------------------------------------------- |
| [SvelteKit 2](https://kit.svelte.dev/)                          | サーバーサイドレンダリングおよびルーティングフレームワーク |
| [Svelte 5](https://svelte.dev/)                                 | コンパイラベースのUIフレームワーク                         |
| [TypeScript](https://www.typescriptlang.org/)                   | 型安全性と開発体験の向上                                   |
| [Tailwind CSS 4](https://tailwindcss.com/)                      | ユーティリティファーストなスタイリング                     |
| [Svelte Transitions](https://svelte.dev/docs/svelte/transition) | 滑らかなページ遷移と複雑なアニメーション（Svelte標準機能） |
| [Lucide Svelte](https://lucide.dev/)                            | シンプルで美しいアイコン                                   |
