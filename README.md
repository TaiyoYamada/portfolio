## 概要

研究・開発経験・作品・受賞歴・技術記事など、私の活動をまとめた個人ポートフォリオサイトです。

### 主な掲載内容

- **About Me**: プロフィールと自己紹介
- **Skills**: 開発スキルとテクノロジースタック
- **Projects**: 個人プロジェクトの紹介
- **Awards**: 受賞歴と実績
- **Articles**: Qiita記事
- **Experience**: 職務経験とインターンシップ
- **Contact**: 連絡先とSNSリンク
